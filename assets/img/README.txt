Chatbot mascot
==============

Save the cat mascot image in this folder as:

    chatbot-cat.png

Requirements:
- PNG with a transparent background
- Roughly square (the source image is ~1024 x 1024)

Once the file is here, it becomes the chat launcher bubble (bottom-right of every
page) and the avatar in the chat header. Until the file exists, the widget falls
back to a chat icon and a penguin, so nothing looks broken.

Remember to copy the same file into the deploy mirror as well:
    ipartment-vn/assets/img/chatbot-cat.png
